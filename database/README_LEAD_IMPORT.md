# Import leads from sheet into Client to Lead

Use this to bulk-import leads from your Google Sheet / Excel export into **Lead** and **Closed Leads** (same `public.leads` table; **Status** = Open or Closed).

## Steps

1. **Export your sheet as TSV** (tab-separated):
   - Keep the first row as the stage names (Lead, Contacted, Brochure, …).
   - Second row must be the **header** with columns:  
     `Company Name`, `Status`, `Stage`, `Assigned POC`, `Person Name`, `Mobile Number`, `Email ID`, `Lead Generation Date`, `Lead Generated By`, `Contacted Date`, `Relevant?`, `City`, `State`, … `Last Contacted On`, `Remarks`.
   - Save as `leads_sheet.tsv` in the `database` folder (or paste that content into a file).

2. **Generate the SQL** (from repo root or `database` folder):
   ```bash
   cd database
   python gen_lead_import_sql.py leads_sheet.tsv > IMPORT_LEADS_FROM_SHEET.sql
   ```
   Or with stdin:
   ```bash
   python gen_lead_import_sql.py < leads_sheet.tsv > IMPORT_LEADS_FROM_SHEET.sql
   ```

3. **Run the SQL in Supabase**:
   - Open **Supabase Dashboard → SQL Editor**.
   - Paste the contents of `IMPORT_LEADS_FROM_SHEET.sql`.
   - Run the script.

## What the script does

- **Reference number**: Auto-generated as `IMP0001`, `IMP0002`, … (sequence `lead_import_ref_seq`).
- **Timestamp** (`created_at`): Taken from **Lead Generation Date** (DD/MM/YYYY); if missing, uses current time.
- **Status**: Uses **Status** column → `Open` or `Closed` (so rows appear in Lead list or Closed Leads accordingly).
- **Stage**: Uses **Stage** column as-is (e.g. Demo Completed, Quotation, Lead, Contacted, Brochure).
- **Assigned POC**: Resolved from **Assigned POC** by matching `full_name` in `public.user_profiles` (e.g. Rimpa Manik, Shubham Kejriwal, Shreyasi Mukherjee). If no match, `assigned_poc_id` is left NULL.
- **Stage data**: Fills `public.lead_stage_data` with JSONB for each stage (lead_details, contacted, brochure, demo_schedule, demo_completed, offer_letter, po, performa_invoice, implementation_invoice, whatsapp_group, setup_data, account_setup, item_setup, first_invoice, first_invoice_payment, final_closing) from the corresponding sheet columns.

## Sample

A sample TSV with 3 rows is in `leads_sheet_sample.tsv`. To generate sample SQL:

```bash
python gen_lead_import_sql.py leads_sheet_sample.tsv > IMPORT_LEADS_FROM_SHEET_sample.sql
```

Then run `IMPORT_LEADS_FROM_SHEET_sample.sql` in Supabase to test.

## Requirements

- `public.leads` and `public.lead_stage_data` (from `CLIENT_TO_LEAD_LEADS.sql`).
- `public.user_profiles` with `full_name` populated for Assigned POC names used in the sheet.
